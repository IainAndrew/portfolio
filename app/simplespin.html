Simplespin.css

Simplespin.css is a CSS animation library I built one evening after work, using Less. It's a lightweight library of rotation effects which can be called easily in your HTML, using short, semantic classes.

My Ambition

To create something that I (and hopefully others) could use to improve my workflow. I have since started using mixin libraries to achieve this, but for those writing pure CSS, I think Simplespin.css can be quite helpful.
