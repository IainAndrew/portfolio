Peter Ashmore

I created this single page site for Dublin based audio engineer & producer Peter Ashmore to promote his services. This was a freelance project which was a lot of fun to work on.

My Ambition

To create a neat, easy-to-navigate page to supply the visitor with all the information they need about Peter in one place.
The client needed an embedded audio player as well as an Instagram feed - this was an opportunity for me to try something new and make them as visually pleasing as possible.

I was approached by Peter to build a simple single page site to promote himself and showcase his skills. I was given a lot of creative freedom with this project, and wanted to create something intuitive and useful, as well as attractive. He requested a clean, flat, modern design with a striking colour palette.

I love subtle visual quirks - and saw an opportunity to implement this cool little feature on the site. The image was drawn out in Illustrator. I used the jQuery .scroll() method to trigger CSS transforms on the tape reels, based on the scroll position.

The tape machine was drawn in Illustrator using the pen and shape tools, then painted with the live paint tool, and finally each part was exported to SVG.

Peter requested an embedded audio player to showcase examples of his work. For this, I used the excellent Toneden Player which links to Soundcloud, and modified it with CSS to my liking. The player features a really nice SVG waveform visualiser.

An Instragram feed was also needed for this project. I decided to use Instafeed.js for this, a javascript plugin that utilises the Instagram API. This worked perfectly for my needs. I also used a custom JS template within Instafeed to show the number of likes and comments on hover.
